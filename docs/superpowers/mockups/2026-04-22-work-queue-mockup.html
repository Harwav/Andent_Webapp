Why This Layout Changes

The page now uses section breaks instead of filter chips. The visual separation tells the operator what still needs attention and what is already moving through backend work, while keeping the same table language in both places.

File Analysis Upper section stays close to the current Active tab and remains action-oriented.
In Progress Lower section mirrors the same columns, but the placement signals "watch only, no action needed."
Status Cell Progress still lives inside the existing Status cell rather than adding a new column.
